# DriverLoaderPro

DriverLoaderPro 是一个windows内核驱动加载工具

- 开源(bullshit)
- 支持微过滤驱动加载 自动写注册表项
- 支持winXP-win10 兼容x86和x64
- 支持输入控制码
- 支持简单的输入字符串 (ascii unicode hex)
- 支持输出显示(ascii unicode hex)
- 友好的UI和提示(手动滑稽)
